# Privacy Policy (GitHub Pages)

This folder is published via GitHub Pages for the Play Store privacy policy requirement.

## Setup

1. **Enable GitHub Pages** on the Miba repo:
   - Settings → Pages
   - Source: **Deploy from a branch**
   - Branch: **main**, Folder: **/docs**
   - Save

2. **URL** (after deployment):
   - `https://<username>.github.io/Miba/privacy-policy.html`

3. **Add to Play Console**: Policy → App content → Privacy policy → paste the URL above
